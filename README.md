# 🌐 My Portfolio

A personal portfolio website built with **React**, **Tailwind CSS**, and **DaisyUI**, showcasing my projects, skills, and experiences.  
All images and assets are served via **Cloudinary** for optimized performance.

---

## 🚀 Tech Stack

- **React** – Frontend library  
- **Tailwind CSS** – Utility-first CSS framework  
- **DaisyUI** – Tailwind component library for UI elements  
- **Cloudinary** – Image hosting & optimization  

---

## ✨ Features

- Responsive design (mobile-friendly 📱)  
- Modern UI with DaisyUI components  
- Fast image loading with Cloudinary CDN  
- Sections for About, Skills, Projects, and Contact
